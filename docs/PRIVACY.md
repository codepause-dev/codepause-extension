# Privacy & Data

> Part of [CodePause](../README.md) documentation

## What We Track

**Tracked (all local):**
- Review time for AI suggestions
- Lines of code (AI vs manual)
- File paths (can be anonymized)
- Review quality metrics
- Session statistics

**NOT tracked:**
- Actual code content
- Keystrokes
- Credentials
- What you type
- Proprietary information

## Data Storage

**100% Local:**
- SQLite database in `~/.vscode/codepause/`
- No external servers
- No telemetry
- You own your data

**Data Portability:**
- Export to JSON/CSV anytime
- `CodePause: Export Data`
- Anonymized exports available
